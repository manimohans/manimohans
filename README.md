## üöÄ Featured Projects

### [**Local LLM Helper**](https://github.com/manimohans/obsidian-local-llm-helper) ![Stars](https://img.shields.io/github/stars/manimohans/obsidian-local-llm-helper?style=social)
Privacy-first Obsidian plugin integrating local LLMs (Ollama, OpenAI, LM Studio). Features text summarization, professional tone adjustment, action item generation, and an interactive chat interface with your notes.

**Tech:** TypeScript, Ollama, LlamaIndex

### [**Lenny's Roundtable**](https://github.com/manimohans/lennysroundtable)
RAG-powered app featuring **300+ transcripts** from Lenny's Podcast. Ask questions and watch 292 product leaders from Stripe, Netflix, Airbnb discuss answers based on what they actually said. Multi-round discussions with semantic search.

**Tech:** Python, LlamaIndex, ChromaDB, Streamlit

### [**Framework Fan RGB**](https://github.com/manimohans/framework_fan_rgb)
Lightweight Rust utility for controlling RGB lighting on Framework Desktop fans. Features both a CLI for scripting and an intuitive GUI with color pickers and creative presets (Stack Overflow Rainbow, Corporate Compliance Orange, Terminal Green Matrix).

**Tech:** Rust, egui, framework_lib

### [**The Verge News MCP Server**](https://github.com/manimohans/verge-news-mcp) ![Stars](https://img.shields.io/github/stars/manimohans/verge-news-mcp?style=social)
Model Context Protocol server that brings The Verge's RSS feed to Claude Desktop. Fetch daily or weekly tech news, search articles by keyword, and get random news selections.

**Tech:** TypeScript, MCP, RSS

### [**Sidekick Chrome Extension**](https://github.com/manimohans/sidekick-chrome-extension)
Chrome extension for chatting with local LLMs (Ollama, LM Studio) via a browser sidebar. Privacy-focused with no data leaving your machine.

**Tech:** JavaScript, Chrome Extensions API
